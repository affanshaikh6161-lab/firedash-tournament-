# Capacitor Setup Guide - Convert Next.js to Android APK

This guide explains how to convert your Next.js application into an installable Android APK using Capacitor.

## Prerequisites

### System Requirements
- **Node.js**: v18+ 
- **npm**: v9+
- **Java Development Kit (JDK)**: v17+ 
- **Android SDK**: API level 33+
- **Android Studio**: Latest version (for emulator and build tools)

### Environment Setup

1. **Install Java JDK 17+**
   ```bash
   # macOS (using Homebrew)
   brew install openjdk@17
   
   # Windows: Download from https://www.oracle.com/java/technologies/downloads/
   # Linux: sudo apt-get install openjdk-17-jdk
   ```

2. **Set JAVA_HOME environment variable**
   ```bash
   # macOS/Linux
   export JAVA_HOME=$(/usr/libexec/java_home -v 17)
   
   # Windows (PowerShell)
   $env:JAVA_HOME = "C:\Program Files\Java\jdk-17"
   ```

3. **Set ANDROID_HOME environment variable**
   ```bash
   # macOS/Linux
   export ANDROID_HOME=$HOME/Library/Android/sdk
   export PATH=$PATH:$ANDROID_HOME/emulator
   export PATH=$PATH:$ANDROID_HOME/tools
   export PATH=$PATH:$ANDROID_HOME/tools/bin
   export PATH=$PATH:$ANDROID_HOME/platform-tools
   
   # Windows
   $env:ANDROID_HOME = "$env:LOCALAPPDATA\Android\sdk"
   $env:PATH = "$env:PATH;$env:ANDROID_HOME\tools;$env:ANDROID_HOME\platform-tools"
   ```

## Step-by-Step Build Instructions

### Step 1: Install Dependencies
```bash
npm install
```

### Step 2: Build Next.js for Export
```bash
npm run build
```
This creates a static `out/` directory that Capacitor will use.

### Step 3: Initialize Android Platform
```bash
npx cap add android
```
This creates an `android/` directory with the Android project structure.

### Step 4: Sync Capacitor Configuration
```bash
npx cap sync android
```
This copies your web assets to the Android project.

### Step 5: Build APK

#### Option A: Using Gradle (Command Line)
```bash
npm run cap:build-apk
```

Or manually:
```bash
cd android
./gradlew assembleDebug
cd ..
```

The APK will be at: `android/app/build/outputs/apk/debug/app-debug.apk`

#### Option B: Using Android Studio (Recommended for first-time builds)
```bash
npx cap open android
```
Then in Android Studio:
1. Click **Build** menu
2. Select **Build Bundle(s) / APK(s)** → **Build APK(s)**
3. Wait for the build to complete

### Step 6: Extract APK
After building, the debug APK is located at:
```
android/app/build/outputs/apk/debug/app-debug.apk
```

## Installation on Device/Emulator

### Using Emulator
```bash
# List available emulators
emulator -list-avds

# Start an emulator
emulator -avd <emulator_name>

# Install APK on emulator
adb install android/app/build/outputs/apk/debug/app-debug.apk
```

### Using Physical Device
1. Enable **Developer Mode** on your Android device
2. Enable **USB Debugging** in Developer Options
3. Connect device via USB
4. Install APK:
   ```bash
   adb install android/app/build/outputs/apk/debug/app-debug.apk
   ```

### Or use Android Studio
1. Open Android Studio
2. Go to **Build** → **Select Build Variant**
3. Choose **debug**
4. Click **Run** button

## Creating Release APK

For production release:

```bash
# Build release APK
cd android
./gradlew assembleRelease
cd ..
```

The release APK is at: `android/app/build/outputs/apk/release/app-release.apk`

**Note**: You'll need to:
1. Create a keystore file
2. Sign the APK
3. Update `android/app/build.gradle` with signing config

## Troubleshooting

### Issue: "gradle wrapper not found"
```bash
# Make gradlew executable on macOS/Linux
chmod +x android/gradlew
```

### Issue: "JAVA_HOME is not set"
```bash
# Verify Java installation
java -version

# Set JAVA_HOME explicitly before building
export JAVA_HOME=$(/usr/libexec/java_home -v 17)
```

### Issue: "Android SDK not found"
```bash
# Install Android SDK via Android Studio or:
sdkmanager --list
sdkmanager "platform-tools"
sdkmanager "platforms;android-33"
sdkmanager "build-tools;34.0.0"
```

### Issue: "Gradle build fails"
```bash
# Clean build
cd android
./gradlew clean
./gradlew assembleDebug
cd ..
```

## Publishing to Google Play Store

After creating a release APK:

1. **Create Google Play Developer Account** ($25 one-time fee)
2. **Generate Keystore** for signing releases
3. **Update build.gradle** with signing configuration
4. **Create App** on Google Play Console
5. **Upload APK** and fill in store details
6. **Submit** for review

See: https://capacitorjs.com/docs/android/deploying-updates


## Configuration Reference

- **capacitor.config.ts**: Main Capacitor settings
- **android/app/src/main/AndroidManifest.xml**: App permissions and settings
- **android/app/build.gradle**: Android build configuration
- **next.config.ts**: Next.js build configuration (must have `output: 'export'`)

## Performance Tips

1. **Optimize bundle size**: Run `npm run build` and check the output size
2. **Enable ProGuard** in release builds to reduce APK size
3. **Use lazy loading** for React components
4. **Minimize image sizes** before adding to project
