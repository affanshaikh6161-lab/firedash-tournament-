# **App Name**: FireDash

## Core Features:

- Match Calendar: A clean, interactive schedule displaying upcoming Free Fire tournaments with real-time countdowns.
- One-Tap Entry: Streamlined join functionality allowing players to register for tournaments instantly via their profile.
- AI Narrative Engine: A dynamic tool that uses AI to generate hype-filled match descriptions and rivalry previews based on team statistics.
- Command Hub: A secure admin dashboard to create, update, and manage match results and participants.
- Dynamic Leaderboard: An automated visual ranking system to track top-performing players and teams across the league.
- Player Dossiers: Visual profile cards detailing player career stats, preferred loadouts, and tournament history.

## Style Guidelines:

- The interface uses a Dark color scheme, inspired by high-stakes digital environments.
- Primary color: Electric Amethyst (#9D5CFF), chosen for its vibrant and modern energy.
- Background color: Midnight Slate (#14111A), providing high contrast for the neon-inspired palette.
- Accent color: Cerulean Cobalt (#5C67FF), used for interactive elements and subtle gradients to drive focus.
- Typography: A pairing of 'Space Grotesk' for tech-oriented headlines and 'Inter' for clear, highly-legible body text.
- Minimalist, sharp-lined icons with neon glows to maintain the precision-gaming aesthetic.
- Modular grid-based layout that prioritizes quick scanability for match dates and rankings.
- Smooth, subtle CSS transitions for hover states and sleek drawer animations for the admin dashboard.
